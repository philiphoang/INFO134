Introduction

The data used in this project are modified copies of Norges befolkning from website https://www.ssb.no/.

Pages

  • Overview: List over all the municipalities (as of 2018) with population data.
  • Details: Get more detailed data of a given municipality such as population, education and employement.
  • Comparing: Compare education levels of two given municipalities and see which municipality comes on top

Link to official datasets:

  • Population: Dataset nr. 104857
  • Employement: Dataset nr. 100145
  • Education: Dataset nr. 85432

Note that the data used in this project might not be accurate as they are modified copies.

The datasets are different from eachother as education dataset contains municipalities before 2018 in contrast to the two others.
The two datasets do not have data on municipalities before 2017 and will not be accessible in page Overview and Details.

Page Comparing is the only page that is able to show municipalities before 2018.

Website created in the context of INFO134 Semester thesis.
